Cambios finales:

1. Exportación de ingresos con formato tipo reporte:
   - Título combinado en español y chino.
   - Secciones por concepto.
   - Encabezados Ingreso / 收入, Monto / 金额, Observación / 备注.
   - Colores amarillo y verde como el ejemplo.

2. Exportación matricial con formato:
   - Título combinado en español y chino.
   - Encabezados Propiedad / 物业 y meses español/chino.
   - Celdas verde = pagado, naranja = pendiente, amarillo = parcial.
   - Valores tipo pagado/total, por ejemplo 100/100 o 0/100.

Nota: para poder guardar colores en Excel se cambió la librería XLSX por xlsx-js-style desde CDN.
